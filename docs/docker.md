## 1. __ollama__ (045f7d6b3f85)

- __Образ__: `ollama/ollama:latest`

- __Порт__: 11434:11434

- __Назначение__: Сервис для работы с LLM (Large Language Models)

- __Что делает__:

  - Загружает и обслуживает модели ИИ (phi3:mini, nomic-embed-text)
  - Обрабатывает запросы на генерацию текста
  - Создает векторные эмбеддинги для документов
  - Это официальный образ от разработчиков Ollama
